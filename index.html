my portfolio

Jubilee Austen Website

HTML 5 & CSS Website

Here is an example of a one page static website that I completed. This project gave me practice in using a large hero image, as well as different colors of fonts and backgrounds. It has a working navigation menu and contact images that link to the appropriate website.

Rogue Pickings Project

HTML & CSS from Photoshop Design Comp

This is an example of a project that I coded and styled from a Photoshop design comp. As you can see it has a nice image, some descriptive text about the food truck, a menu, a review and some contact info. The original version of this project gave me practice in using borders, padding, and floating elements. I later revised the project and used flexbox to make it responsive.

Recipe Cards Collection

HTML 5 & CSS Exercise

This example is for a set of three recipe cards. I used CSS to float the images next to the instructions for each recipe. This project used a lot of padding, margins, and borders to make everything line up correctly and evenly. It also has both unordered and ordered lists to keep all the text neatly displayed and readable in a small area.

Unplugged Retreat Multi-Page Site

Responsive Website from Photoshop Design Comp

This last example is a more complex project in responsive coding. It was written with the mobile first approach with added tablet and desktop media queries. It also has two seperate pages that are linked in the nav bar in the header. Again, I used flexbox instead of floating my elements. It gave me a lot of practice in making a more user friendly website that's able to be viewed on any screen size.
